# Hiromasa Portfolio

ポートフォリオサイトです。

## URL

https://xxxxx.vercel.app

## 概要

就職活動用に制作したポートフォリオサイトです。

プロフィールや制作物を掲載し、
自分のスキルや制作経験を分かりやすく伝えることを目的としています。

## 使用技術

### Frontend

- Next.js 15
- React
- TypeScript
- Tailwind CSS

### Infrastructure

- Vercel

### Version Control

- Git
- GitHub

## 主な機能

- Homeページ
- プロフィール表示
- 制作物一覧
- お問い合わせページ
- レスポンシブ対応

## 工夫した点

### コンポーネント設計

共通レイアウトをLayoutコンポーネントとして切り出し、
保守性を向上させました。

### UI/UX

シンプルなデザインを意識し、
必要な情報へ素早くアクセスできる構成にしました。

### GitHub運用

個人開発ではありますがチーム開発での運用を意識し、
Issue駆動で開発を行い、
Projectsを用いてタスク管理を実施しました。
